# 📊 Infografía Interactiva: Control Sanitario en Trucha Arcoíris

Infografía web estática e interactiva diseñada para visualizar datos estadísticos sobre la producción de trucha arcoíris, control sanitario y buenas prácticas acuícolas. El proyecto prioriza un diseño minimalista que revela información compleja únicamente mediante la interacción del usuario.

---

## 📝 Descripción General

Esta aplicación web presenta un conjunto de 10 tarjetas estadísticas que ocultan su contenido detallado (cifras, porcentajes y gráficos de barras) hasta que el usuario interactúa con ellas mediante clic o cursor.

El objetivo visual es transmitir conceptos de ciencia, agua y biología marina, utilizando una paleta de colores fría y tecnológica. No requiere frameworks ni librerías externas, funcionando directamente en el navegador.

---

## 🛠️ Tecnologías Utilizadas

El desarrollo se ha realizado utilizando únicamente tecnologías web nativas (Vanilla), sin dependencias externas:

*   **HTML5:** Estructura semántica (`<article>`, `<header>`, `<main>`, `<footer>`) para accesibilidad y SEO.
*   **CSS3:**
    *   Diseño **Responsive** mediante CSS Grid y Flexbox (enfoque Mobile First).
    *   **Variables CSS** para la gestión de la paleta de colores.
    *   Transiciones y transformaciones para animaciones suaves de tarjetas y gráficos.
*   **JavaScript (ES6+):**
    *   Manipulación del DOM.
    *   Gestión de eventos (click, hover, keypress).
    *   Lógica para la animación dinámica de gráficos de barras.
*   **Tipografía:** Importación de *Montserrat* y *Open Sans* desde Google Fonts.

---

## 📂 Estructura del Proyecto

El proyecto está contenido principalmente en un único archivo HTML para facilitar su despliegue inmediato. Sin embargo, la estructura interna está organizada para facilitar la separación de responsabilidades.

```text
/mi-proyecto-trucha
│
├── index.html          # Contiene toda la estructura, estilos y scripts.
├── README.md           # Documentación del proyecto.
└── /assets             # Carpeta sugerida para imágenes futuras.
```

> **Nota:** Aunque todo está en `index.html`, el código está claramente comentado para separar el contenido en `style.css` y `script.js` si se desea modularizar.

---

## 💻 Cómo visualizar localmente

Dado que es un proyecto estático (sin backend), no requiere instalación de Node.js, servidores de base de datos ni compiladores.

### Paso 1: Descargar/Copiar el código
1.  Crea una carpeta en tu computadora llamada `trucha-infografia`.
2.  Dentro de ella, crea un archivo llamado `index.html`.
3.  Copia y pega el código completo proporcionado en el paso anterior dentro de `index.html`.

### Paso 2: Ejecutar
Simplemente haz **doble clic** sobre el archivo `index.html`.

El proyecto se abrirá automáticamente en tu navegador predeterminado (Chrome, Firefox, Edge, Safari). No es necesario estar conectado a internet, salvo para la carga inicial de las fuentes de Google.
